# ČSN 73 4301 — Obytné budovy

## Minimální plochy místností

| Místnost | Min. plocha |
|----------|------------|
| Obývací pokoj | 16m² |
| Ložnice (dvoulůžko) | 12m² |
| Ložnice (jednolůžko) | 8m² |
| Kuchyň | 5m² |
| Kuchyňský kout | 3.6m² |
| Koupelna | 4m² |
| WC | 0.9 × 1.2m |
| Předsíň/chodba | šířka min. 1.1m |

## Světlá výška

- Obytné místnosti: min. 2.5m
- Podkroví: min. 2.3m na ≥50 % podlahové plochy
- Sklep, technická místnost: min. 2.1m
- Garáž: min. 2.1m (pod překladem)

## Oslunění a osvětlení

- Obytné místnosti musí mít přímé denní osvětlení (okna)
- Plocha oken: min. 1/8 podlahové plochy místnosti (dříve 1/10)
- Min. 1 obytná místnost musí být osluněna min. 90 minut dne 1. března

## Schodiště

- Šířka ramene: min. 900mm (rodinný dům), 1100mm (bytový dům)
- Podesta: min. 900 × 900mm
- Zábradlí: výška min. 900mm (do 3m), 1000mm (nad 3m)
- Sklon: max. 35° (pohodlné), přípustný 41° (strmé)

## Větrání

- Každá obytná místnost: otevíratelné okno nebo nucené větrání
- Koupelna/WC bez okna: nucené větrání (min. 50m³/h koupelna, 25m³/h WC)
