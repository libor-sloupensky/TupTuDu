# Tepelná ochrana budov

## Doporučené hodnoty součinitele prostupu tepla U [W/m²K]

| Konstrukce | Doporučená | Požadovaná |
|-----------|-----------|-----------|
| Obvodová stěna | 0.25 | 0.30 |
| Střecha (šikmá) | 0.16 | 0.24 |
| Střecha (plochá) | 0.16 | 0.24 |
| Podlaha na zemině | 0.30 | 0.45 |
| Okna | 1.20 | 1.50 |
| Vstupní dveře | 1.20 | 1.70 |

## Praktické důsledky pro návrh

- Obvodová stěna zděná: min. tloušťka 0.38–0.44m (Porotherm/Ytong)
- Nebo tenčí zdivo (0.25m) + kontaktní zateplení (0.15m EPS/minerální vata)
- Střecha: izolace min. 0.20–0.30m (minerální vata mezi/pod krokve)
- Podlaha: izolace min. 0.10–0.15m (EPS/XPS)
- Okna: trojskla (U ~0.7–0.9) doporučena pro nové stavby
